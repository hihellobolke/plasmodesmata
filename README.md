> 原生質絲（英语：Plasmodesmata）為植物細胞和部分藻類細胞壁間貫穿細胞壁的特有孔道，可以讓相鄰細胞的細胞質相互流通。有微小孔道，為細胞間物質運輸與信息傳遞的重要通道，通道中有一連接兩細胞內質網的連絲微管，細胞質可經由原生質絲交流及運輸，此過程稱為共質體運輸。
